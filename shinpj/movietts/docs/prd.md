# MovieTTS - PRD (Product Requirements Document)

## 1. 제품 개요

| 항목 | 내용 |
|------|------|
| 제품명 | MovieTTS |
| 목적 | 영상의 음성을 분석하여 타임스탬프 기반 자막을 자동 생성 |
| 대상 사용자 | 영상 제작자, 콘텐츠 크리에이터, 교육 콘텐츠 제작자 |
| 플랫폼 | 웹 애플리케이션 |

## 2. 배경 및 문제 정의

### 문제
- 영상 자막 제작은 수작업으로 진행되어 시간과 비용이 많이 소요됨
- 타임스탬프를 수동으로 맞추는 작업이 반복적이고 비효율적
- 기존 자동 자막 도구는 정확도가 낮거나 사용이 복잡함

### 해결 방안
- AI 기반 음성 인식으로 자동 자막 생성
- 정밀한 타임스탬프 자동 동기화
- 직관적인 웹 UI로 업로드부터 다운로드까지 원스톱 제공

## 3. 사용자 페르소나

### 페르소나 A - 유튜브 크리에이터
- 매주 2~3개의 영상을 업로드
- 자막 작업에 영상당 1~2시간 소요
- 한국어 영상에 한국어 자막 필요

### 페르소나 B - 교육 콘텐츠 제작자
- 강의 영상에 정확한 자막 필수
- 자막 편집/수정 기능 중요
- 다양한 자막 포맷 필요 (SRT, VTT)

## 4. 기능 요구사항

### P0 (MVP 필수)

| ID | 기능 | 설명 | 인수 조건 |
|----|------|------|----------|
| F-001 | 영상 업로드 | 사용자가 영상 파일을 업로드 | MP4, AVI, MKV, MOV, WebM 지원, 최대 500MB |
| F-002 | 음성 추출 | 영상에서 오디오 트랙 분리 | FFmpeg 기반, WAV 변환 성공률 100% |
| F-003 | 음성→텍스트 변환 | STT 엔진으로 텍스트 변환 | 타임스탬프 포함, 언어 자동 감지 |
| F-004 | SRT 자막 생성 | 표준 SRT 포맷 자막 파일 생성 | 밀리초 단위 타임스탬프 정확도 |
| F-005 | 자막 다운로드 | 생성된 자막 파일 다운로드 | SRT 파일 다운로드 가능 |

### P1 (확장 기능)

| ID | 기능 | 설명 | 인수 조건 |
|----|------|------|----------|
| F-006 | 다중 포맷 지원 | VTT, ASS 포맷 추가 | 포맷 선택 후 다운로드 |
| F-007 | 자막 편집 | 웹 UI에서 자막 텍스트 수정 | 인라인 편집, 실시간 저장 |
| F-008 | 자막 미리보기 | 영상 플레이어에서 자막 확인 | 영상과 자막 동기화 재생 |
| F-009 | 처리 상태 표시 | 진행률 실시간 표시 | 업로드→추출→변환→완료 단계 표시 |

### P2 (향후 확장)

| ID | 기능 | 설명 |
|----|------|------|
| F-010 | 다국어 번역 | 원본 자막을 다른 언어로 번역 |
| F-011 | 화자 분리 | 여러 화자를 구분하여 자막 표시 |
| F-012 | 배치 처리 | 여러 영상 동시 업로드 및 처리 |

## 5. 사용자 플로우

```
[영상 업로드] → [처리 대기 (진행률 표시)] → [자막 결과 확인]
                                                  │
                                        ┌─────────┼─────────┐
                                        │         │         │
                                    [편집]    [미리보기]  [다운로드]
                                        │                    │
                                    [저장] ──────────→ [포맷 선택]
                                                         │
                                                   [SRT/VTT/ASS]
```

## 6. 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 성능 | 10분 영상 기준 처리 시간 3분 이내 |
| 파일 크기 | 최대 500MB 업로드 지원 |
| 정확도 | 한국어 기준 STT 정확도 90% 이상 |
| 동시 사용자 | 최소 10명 동시 처리 |
| 브라우저 | Chrome, Edge, Safari 최신 버전 지원 |

## 7. 성공 지표 (KPI)

| 지표 | 목표 |
|------|------|
| 자막 생성 성공률 | 95% 이상 |
| 평균 처리 시간 | 영상 길이의 30% 이내 |
| 사용자 자막 수정 비율 | 30% 이하 (높은 초기 정확도) |
| 사용자 만족도 | 4.0/5.0 이상 |
