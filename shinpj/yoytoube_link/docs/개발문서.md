# YouTube 트렌드 분석기 개발 문서

## 1. 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | HTML + CSS + Vanilla JS (또는 React) |
| 백엔드 | Node.js + Express |
| 외부 API | YouTube Data API v3 |
| 캐싱 | 메모리 캐시 (node-cache) |
| 패키지 매니저 | npm |

## 3. 환경 설정

### 3.1 YouTube Data API v3 키 발급
1. [Google Cloud Console](https://console.cloud.google.com) 접속
2. 새 프로젝트 생성
3. `YouTube Data API v3` 활성화
4. 사용자 인증 정보 → API 키 생성

### 3.2 .env 파일
```
YOUTUBE_API_KEY=여기에_API_키_입력
PORT=3000
```

### 3.3 설치 및 실행
```bash
npm install
node server.js
# http://localhost:3000 접속
```

## 4. API 설계

### 4.1 백엔드 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/trending` | 전체 분야 트렌드 요약 (분야별 TOP 3) |
| GET | `/api/category/:id` | 특정 분야 인기 영상 목록 |
| GET | `/api/video/:id` | 영상 상세 분석 |

### 4.2 쿼리 파라미터

**GET /api/category/:id**
| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `limit` | number | 10 | 가져올 영상 수 (최대 100) |
| `order` | string | viewCount | 정렬 기준 (viewCount / rating / date) |
| `pageToken` | string | - | 다음 페이지 토큰 |

### 4.3 응답 형식 예시

**GET /api/category/10 (음악)**

## 5. YouTube Data API 호출 구조

### 5.1 분야별 인기 영상 목록
```
GET https://www.googleapis.com/youtube/v3/videos
  ?part=snippet,statistics,contentDetails
  &chart=mostPopular
  &videoCategoryId={categoryId}
  &regionCode=KR
  &maxResults=50
  &pageToken={pageToken}
  &key={API_KEY}
```

### 5.2 채널 구독자 수 조회
```
GET https://www.googleapis.com/youtube/v3/channels
  ?part=statistics
  &id={channelId}
  &key={API_KEY}
```

> **주의**: API 일일 할당량은 10,000 유닛. 캐싱으로 호출 최소화 필요.

## 6. 캐싱 전략

| 캐시 키 | TTL | 설명 |
|---------|-----|------|
| `trending` | 1시간 | 메인 트렌드 요약 |
| `category:{id}:{order}:{page}` | 1시간 | 분야별 목록 |
| `video:{id}` | 30분 | 영상 상세 정보 |
