# YouTube 분야별 트렌드 분석 사이트 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | YouTube 트렌드 분석기 |
| 목적 | 유튜브 분야별 인기 영상을 좋아요·구독자·조회수 기준으로 분석 제공 |
| 대상 사용자 | 유튜브 트렌드가 궁금한 일반 사용자, 콘텐츠 크리에이터 |

---

## 2. 핵심 기능

### 2.1 메인 화면 — 분야별 트렌드 요약
- 유튜브 카테고리별(음악, 게임, 스포츠, 교육 등) 인기 영상 미리보기
- 각 분야의 TOP 3 영상을 카드 형태로 표시
- 조회수·좋아요·구독자수 수치 표시

### 2.2 분야 선택 — 인기 영상 목록
- 분야 선택 시 기본 10개 표시
- [더보기] 버튼으로 최대 100개까지 로드
- 정렬 기준 선택: 조회수 / 좋아요 / 업로드일

### 2.3 영상 상세 분석
- 영상 클릭 시 상세 분석 페이지 이동
- 분석 항목:
  - 조회수 / 좋아요 / 댓글 수
  - 좋아요율 (좋아요 / 조회수 × 100)
  - 채널 구독자 수
  - 영상 길이 대비 조회수 효율
  - 업로드 후 경과 시간

---

## 3. 화면 구성 (페이지 흐름)

```
메인 화면
 └─ 분야 선택
      └─ 분야별 인기 영상 목록 (10개 → 더보기 → 100개)
           └─ 영상 클릭
                └─ 영상 상세 분석 페이지
```
