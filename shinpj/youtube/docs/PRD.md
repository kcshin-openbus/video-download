# PRD - 유튜브 자막 자동 번역 업로드 도구

## 1. 개요

유튜브 영상의 한국어 자막을 기반으로 다국어 자막을 자동 번역·생성하여 유튜브에 업로드하는 자동화 프로그램.
기존에 사람이 수동으로 각 나라별 자막을 입력하던 작업을 기계가 대신 처리한다.

---

## 2. 배경 및 목적

- 유튜브는 영상별로 언어별 자막을 등록할 수 있으나, 수동 입력은 시간이 많이 소요됨
- 한국어 자막을 기본값으로 두고, 자동 번역을 통해 다국어 자막을 생성하면 작업 효율이 크게 향상됨
- 특정 타임라인 구간(고유명사, 브랜드명, 슬로건 등)은 번역 없이 원문 유지가 필요함

---

## 3. 목표

| 목표 | 설명 |
|------|------|
| 다국어 자막 자동 생성 | 한국어 자막 → 선택한 언어로 자동 번역 |
| 특정 구간 번역 제외 | 타임라인 단위로 번역 예외 처리 지원 |
| 유튜브 자막 자동 업로드 | YouTube Data API를 통해 번역된 자막 자동 등록 |
| 웹 UI 제공 | 비개발자도 쉽게 사용할 수 있는 인터페이스 |

---

## 4. 사용자 시나리오

1. 사용자가 유튜브 영상 URL 또는 영상 ID를 입력한다.
2. 시스템이 해당 영상의 한국어 자막(SRT/VTT)을 가져온다.
3. 사용자가 번역 대상 언어를 선택한다. (예: 영어, 일본어, 중국어 등)
4. 사용자가 번역하지 않을 타임라인 구간을 지정한다. (선택 사항)
5. 시스템이 번역 API를 통해 자막을 번역하고, 예외 구간은 원문 유지한다.
6. 번역된 자막을 미리보기로 확인하고 수정할 수 있다.
7. 확인 후 유튜브에 자막을 자동 업로드한다.

---

## 5. 기능 요구사항

### 5.1 자막 불러오기
- 유튜브 영상 URL 입력으로 한국어 자막 자동 추출
- 로컬 SRT / VTT 파일 직접 업로드도 지원

### 5.2 번역
- 번역 대상 언어 다중 선택 지원 (영어, 일본어, 중국어 간체/번체, 스페인어 등)
- Google Translate API 또는 DeepL API 연동
- 라인 단위 번역 처리

### 5.3 번역 예외 처리
- 타임라인 구간 지정 시 해당 구간의 자막 텍스트를 번역하지 않고 원문 유지
- 특정 단어/구문 글로벌 예외 목록 설정 가능

### 5.4 자막 편집 및 미리보기
- 번역 결과를 UI에서 라인별로 수정 가능
- 원문과 번역문 나란히 표시 (Side-by-side)

### 5.5 유튜브 업로드
- YouTube Data API v3를 통해 번역된 자막 자동 업로드
- 업로드 상태 실시간 피드백

---

## 6. 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 보안 | YouTube OAuth2 인증, API 키 서버 측 관리 |
| 성능 | 1,000줄 이하 자막 기준 번역 30초 이내 완료 |
| 호환성 | Windows / macOS / Linux 지원 (Node.js 기반) |
| 확장성 | 번역 API를 교체 가능한 플러그인 구조 |

---

## 7. 제외 범위 (Out of Scope)

- 음성 인식(STT)을 통한 자막 자동 생성
- 유튜브 이외의 플랫폼(Vimeo, TikTok 등) 지원
- 유료 번역 API 비용 관리 기능
