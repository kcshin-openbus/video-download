# PRD: 외국어 교재 MP3 자동 배달 서비스

## 1. 개요

학습자가 원하는 일본어/중국어 교재의 음원(MP3) 파일을 이메일로 즉시 전송받을 수 있는 웹 서비스.

## 2. 목표

- 학습자가 교재를 검색하고, 이메일만 입력하면 MP3를 자동으로 수신
- 관리자가 교재/음원 데이터를 관리하고 발송 이력을 모니터링
- 최소한의 UX 단계로 빠르고 직관적인 사용 경험 제공

## 3. 타겟 사용자

| 사용자 | 설명 |
|--------|------|
| 학습자 (User) | 일본어/중국어 교재 MP3를 이메일로 받고 싶은 외국어 학습자 |
| 관리자 (Admin) | 교재 데이터 등록/수정 및 발송 현황을 관리하는 운영자 |

## 4. 핵심 기능

### 4.1 사용자 페이지 (User Interface)

#### F1. 지능형 교재 검색
- 2글자 이상 입력 시 관련 교재 리스트를 실시간 필터링
- 교재명, 언어(일본어/중국어), 출판사 등으로 검색 가능
- 검색 결과에 교재 정보(제목, 언어, 출판사, 음원 개수) 표시

#### F2. 교재 선택 및 이메일 입력
- 교재 선택 시 이메일 입력창 활성화
- 이메일 형식 유효성 검증
- 발송 버튼 클릭 또는 입력 완료 즉시 발송 트리거

#### F3. 자동 이메일 발송
- 선택된 교재의 MP3 파일을 첨부하여 이메일 자동 발송
- 발송 상태(진행중/완료/실패) 실시간 피드백
- 발송 완료 시 성공 메시지 표시

### 4.2 관리자 페이지 (Admin Dashboard)

#### F4. 교재 데이터 관리 (CRUD)
- 교재 등록: 제목, 언어, 출판사, MP3 파일 업로드
- 교재 수정/삭제
- 교재 목록 조회 (페이지네이션, 검색, 필터)

#### F5. 메일 발송 이력 모니터링
- 발송 이력 목록: 수신자 이메일, 교재명, 발송 시간, 상태
- 필터링: 날짜, 상태(성공/실패), 교재별
- 발송 통계 대시보드 (일별/주별 발송 건수)

### 4.3 메일 발송 모듈 (Mail Module)

#### F6. 이메일 전송 엔진
- SMTP 또는 이메일 API(Gmail, SendGrid 등) 기반 발송
- MP3 파일 첨부 발송
- 발송 실패 시 재시도 로직
- 발송 결과 DB 기록

## 5. 데이터 모델 (개념)

### 교재 (Textbook)
| 필드 | 설명 |
|------|------|
| id | 고유 식별자 |
| title | 교재명 |
| language | 언어 (일본어/중국어) |
| publisher | 출판사 |
| mp3_files | 음원 파일 목록 |
| created_at | 등록일 |

### 발송 이력 (SendLog)
| 필드 | 설명 |
|------|------|
| id | 고유 식별자 |
| textbook_id | 교재 ID |
| recipient_email | 수신자 이메일 |
| status | 발송 상태 (pending/sent/failed) |
| sent_at | 발송 시간 |
| error_message | 실패 시 오류 메시지 |

## 6. Seed 데이터

테스트를 위해 아래 더미 데이터를 사전 셋팅:

- **일본어 교재 5개 이상**: 제목, MP3 파일 포함
- **중국어 교재 5개 이상**: 제목, MP3 파일 포함

## 7. 비기능 요구사항

- 직관적이고 세련된 UI (AI 디자인 어시스턴트 활용)
- 실시간 검색 응답 시간 < 300ms
- 이메일 발송 완료 시간 < 30초
- 모바일 반응형 지원

## 8. 일정

- **해커톤 기간**: 24시간 집중 개발
- **최종 스크럼**: 시작 24시간 후 시연 및 AI 협업 경험 공유
- **제출**: GitHub 레포지토리 링크 공유
