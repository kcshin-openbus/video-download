 Mission: 외국어 교재 MP3 자동 배달 서비스 구축
학습자가 원하는 일본어/중국어 교재의 음원 파일을 이메일로 즉시 전송받을 수 있는 간결하고 파워풀한 웹 서비스를 구축합니다.

[핵심 기능 프로세스]
지능형 검색: 2글자 이상 입력 시 관련 교재 리스트 실시간 필터링 및 나열
이메일 게이트웨이: 특정 교재 선택 시, 음원을 수령할 이메일 입력창 활성화
자동 발송 시스템: 메일 주소 입력 완료 즉시 해당 교재의 MP3 파일을 첨부하여 발송
[필수 구현 범위]
User Interface: AI 디자인 어시스턴트를 활용한 직관적이고 세련된 사용자 페이지
Admin Dashboard: 교재 데이터 관리 및 메일 발송 이력 모니터링을 위한 관리자 페이지
Mail Module: 실제 메일 전송 로직이 포함된 백엔드 모듈
🛠 Special Tasks (필수 준수 사항)
AI 바이브 코딩 (Primary Rule)
Claude Code를 이용한 바이빙
단순 구현을 넘어 AI를 통한 코드 리뷰 및 최적화 과정을 권장합니다.
Seed 데이터 구성
원활한 테스트를 위해 일본어 및 중국어 교재/MP3 데이터를 각 5개 이상의 더미 데이터로 사전 셋팅하십시오.
결과물 제출 및 공유
